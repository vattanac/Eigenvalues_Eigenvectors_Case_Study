## 🧮 Core Formulas at a Glance

```
Eigenvalue equation:      A · x = λ · x
Characteristic equation:  det(A − λI) = 0
2×2 shortcut:             λ² − (trace)·λ + (det) = 0

Trace  = a₁₁ + a₂₂ + ... = λ₁ + λ₂ + ...
Det(A) = λ₁ · λ₂ · ...

Identity shift:  (A + cI)·x = (λ + c)·x  [same eigenvectors]
```
